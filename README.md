#Customer Churn Prediction

This project predicts whether a customer will leave a company using Machine Learning.

🔍 Project Steps

- Data preprocessing
- Feature selection
- Train-test split
- Model training (Logistic Regression & Random Forest)
- Model evaluation

🤖 Models Used

- Logistic Regression
- Random Forest (Best Performance)

📊 Result

Random Forest achieved higher accuracy compared to Logistic Regression.

📁 Dataset

Churn Modelling Dataset

🚀 Tools Used

- Python
- Pandas
- Scikit-learn
- Google Colab
